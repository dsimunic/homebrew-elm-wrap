# Dsimunic Elm-wrap

## How do I install these formulae?

`brew install dsimunic/elm-wrap/<formula>`

Or `brew tap dsimunic/elm-wrap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "dsimunic/elm-wrap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
